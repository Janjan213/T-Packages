# T-Packages
Termux Packages In one Script

# Requirements:
Download Termux app 
Download Termux-Api

# Installation:
Open Termux app and Type
$ apt update && apt upgrade
$ apt install git

# Clone this url :
$ git clone https://github.com/Janjan213/T-Packages.git

# Change Directory :
$ cd T-Packages
$ chmod +x * ( or ) chmod +x t-packages.sh
# Run
$ sh t-packages.sh

And Done ......
